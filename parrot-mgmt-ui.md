# 鹦鹉管理系统 - UI设计方案

## 1. 设计概述

### 设计理念
- **简洁高效**：界面清晰明了，减少认知负担
- **移动优先**：适配手机端快速操作
- **视觉统一**：一致的视觉语言和交互模式
- **快速响应**：照片预览、状态切换等操作秒级完成

### 颜色方案
- **主色**：翡翠绿 (#10B981) - 代表生命力和活力
- **辅助色**：琥珀金 (#F59E0B) - 温暖和友善
- **状态色**：
  - 未售：翡翠绿 (#10B981)
  - 已售：橙色 (#F97316)
  - 退货：红色 (#EF4444)
- **中性色**：
  - 深灰：#374151 (文字)
  - 中灰：#6B7280 (次要信息)
  - 浅灰：#F9FAFB (背景)

### 字体系统
- **标题**：Noto Sans SC Bold
- **正文**：Noto Sans SC Regular
- **数字**：JetBrains Mono (价格、编号)

### 3.2 照片管理设计

#### 照片上传流程：
1. **上传入口**：
   - 大按钮"+"，文字"点击或拖拽上传"
   - 支持点击选择文件
   - 支持拖拽上传

2. **照片预览**：
   ```
   ┌────────┐
   │        │
   │  图片  │  ★ 设为封面

   └────────┘
   ```

   - 最多5张照片
   - 单张照片不超过5MB
   - 允许格式：JPG、PNG、WEBP

- **相机直拍**：移动端支持直接调用相机拍照
- **多选上传**：一次选择多张照片
- **压缩优化**：前端压缩图片，提升上传速度

## 10. 性能优化建议

### 10.1 图片优化

- **响应式图片**：
  - 缩略图：200x200px
  - 预览图：800x800px
  - 原图：按需加载

- **懒加载**：
  - 列表页使用懒加载，滚动到可视区域再加载图片
  - 减少初始加载时间

### 10.2 虚拟滚动

- 鹦鹉数量超过100只时使用虚拟滚动
- 只渲染可视区域的卡片
- 提升滚动性能

### 10.3 缓存策略

- 图片使用浏览器缓存
- 数据使用IndexedDB缓存
- 支持离线查看已加载数据

  - Ctrl+N: 新增鹦鹉

### 11.3 高对比度
